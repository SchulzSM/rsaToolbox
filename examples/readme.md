# Example Data

This directory contains example datasets for demonstrating and testing the rsaToolbox.

## Structure

The data are organized by participant:

* `participant_01/`
* `participant_02/`

Each participant folder contains multiple CSV files organized into subfolders representing different stages of data processing and analysis.

## Usage

These example datasets can be used to:

* estimate baseline RSA parameters from paced breathing data
* apply respiration-corrected RSA analysis
* test batch processing functionality

## Notes

* The data are provided for demonstration purposes only.
* File formats are compatible with the MATLAB implementation of rsaToolbox.
* Refer to the documentation (`docs/`) for details on required input formats.

## Download

A complete archive of the example data is also available via the GitHub releases for convenience.
