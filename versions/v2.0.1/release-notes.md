# rsaToolbox v2.0.1

## Overview

This release provides an updated MATLAB-based implementation of rsaToolbox, including a compiled Windows executable.

## Contents

* MATLAB source code
* Windows executable (32-bit)
* Example data

## Notes

* Minor update within the 2.x release series.
* Includes incremental improvements and bug fixes over v2.0.0.
* The executable requires the MATLAB Compiler Runtime (MCR).
