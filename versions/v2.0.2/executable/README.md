# Executable – rsaToolbox v2.0.2

This directory documents the executable version of rsaToolbox v2.0.2.

The Windows executable is **not stored directly in the repository**, but is available via the GitHub release for this version.

## Download

Please download the executable from the corresponding GitHub release:

* Release: [v2.0.2](https://github.com/SchulzSM/rsaToolbox/releases/tag/2.0.2)

## Requirements

This executable requires the MATLAB Compiler Runtime (MCR):

* Version: **7.14.1 (MATLAB R2010b)**

The required runtime is **not included** in this repository.

Please obtain the appropriate version from MathWorks:

https://www.mathworks.com/products/compiler/matlab-runtime.html

Note: Older versions of the MATLAB Runtime may not be directly available via the main download page and might require access via archived resources or MathWorks support.

## File integrity

After downloading, verify the integrity of the file using the SHA256 checksum provided in `SHA256SUMS.txt`.

## Notes

* This executable is a 32-bit Windows build.
* This is the **final MATLAB-based release** of rsaToolbox.
* Future development will focus on a Python reimplementation (planned as v3.x).
