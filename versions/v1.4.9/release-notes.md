# rsaToolbox v1.4.9

## Overview

This release provides a stable MATLAB-based implementation of rsaToolbox, including a compiled Windows executable.

## Contents

* MATLAB source code
* Windows executable (32-bit)
* Example data

## Notes

* This version predates the 2.x series and represents an intermediate stable release.
* The executable requires the MATLAB Compiler Runtime (MCR).
